Privacy Policy

Last updated: 2026-06-15

StreamClipper ("the App") is a personal desktop application used solely by its developer ("the User") to monitor their own streaming and social media account statistics.

1. Data Collected

The App may store, locally on the User's own device only:

  • OAuth access tokens and API keys for third-party platforms (Twitch, Kick, YouTube, Instagram, TikTok), entered by the User in the App's Settings.
  • Public and account-level statistics fetched from these platforms for the User's own account (e.g. follower counts, subscriber counts, video view/like/comment counts, stream viewer counts).
  • Local recordings, clips, and metadata generated from the User's own stream sessions.

2. Data Storage

All data is stored locally on the User's own computer, in local configuration and JSON files. No data is transmitted to or stored on any external server controlled by the developer.

3. Data Sharing

No data is sold, shared, or transmitted to any third party other than the platforms (Twitch, Kick, YouTube, Instagram, TikTok) the User has explicitly connected the App to, and only for the purpose of retrieving the User's own account data.

4. Data Retention & Deletion

Since all data is stored locally, the User may delete any or all stored data at any time by removing the relevant local files (e.g. settings file, local history files) or uninstalling the App.

5. Contact

For questions about this policy, contact the developer via the GitHub repository associated with this page.
